Triangle Area Calculator - A Java App

This Java application takes in data of triangle(s) and returns the area of the triangle(s).

How it works

First, download the java application. Next, open terminal. Travel to the location containing the application (typically /Downloads).
Run the program using the command:
java triarea [INPUT HERE]
The input is as follows: the first number is the number of triangles of which you will need the area.
The next six numbers are the x and y cordinates of the vertices of the traingle (x1, y1, x2, y2, x3, y3).The next six numbers are the cordinates of the second triangle, etc. For example, if you are processing the are of 2 triangles your input should be 2 followed by 12 numbers representing the cordinates of both triangles. The output will be the area of the triangles.
