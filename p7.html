Document Analyzer

The following code which can be found here or downloaded here is an SML code designed to analyze a body of text and out each unique word along with the number of times the word appears in the document.

My Approach

The code works with a typical dictionary structure, a key*value pair per entry. In this example, the key is the word and the value is the number of times the word has appeared. The function, f(), runs through the document, inputted as a string list, checking if the string is in the dictionary (hasKey()). If the string is present in the dictinary, the dictionary is updated (update()) by increasing the value of the present key by 1. If the string is not present in the dictionary, the dictionary is updated by adding an entry with the key as the string and the value as 1.

How to Use My Code

Download the First, download the application by clicking here. Next, open terminal. Travel to the location containing the application (typically /Downloads). Run the command: smlnj; Type: use "docanalyzer.sml" to tell shell which document to use. You will see some information about the program, inclduing functions and variables. Use the funcion f() which does the analyzing as you please. Below is the spec for the function:
SPEC: This method, f, takes in a doc : string list and a dictionary : (string * int) list, and outputs a dictionary, [('word', word_count), ('word2', word2_count), etc.], such that it presents every unique word and its word count. When analyzing a document, doc, please run this method either with [] or the variable dict for the dicitonary type in order to recieve a correct analysis. For an example testing, RUN f(example,[]) or f(example,dict)
